# Better Roblox Badges

**Better Roblox Badges** is a browser extension designed for Chromium-based browsers. It therefore supports *Google Chrome*, *Opera*, *Opera GX*, *Brave*, *Microsoft Edge*, *Safari* and *Firefox*. It is designed for computers and may not function on other devices.

It cannot operate outside the browser on which it is installed and therefore cannot modify the *Roblox Player* application.

To function, the extension requires you to be logged into your Roblox account at [roblox.com](https://roblox.com).
